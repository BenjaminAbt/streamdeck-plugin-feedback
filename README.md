# Stream Deck Plugins - Feedback & Support

Welcome 👋

This repository is the central place for **feedback, bug reports, and feature suggestions** for my Stream Deck plugins.

If something is not working as expected, you have an idea for improvement, or you just want to share your experience - this is the right place.

---

## How to Report an Issue

Please use one of the provided issue forms:

* 🐞 **Bug Report**
  If something does not work as expected

* 💬 **Feedback**
  Share your experience, suggestions, or ideas

Using the forms helps collect the information needed to investigate issues quickly.

👉 Open a new issue here: [Feedback and Issues](https://github.com/BenjaminAbt/streamdeck-plugin-feedback/issues)

---

## Before Opening an Issue

Please check:

* you are using the **latest plugin version**
* you are running the **latest Stream Deck software**
* the issue still occurs after restarting Stream Deck

This helps avoid common problems and speeds up support.
